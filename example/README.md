## Get Start

```sh
git clone https://github.com/alantoa/react-native-reanimated-player.git
cd react-native-reanimated-player
yarn bootstrap
```

### iOS

```sh
cd example
yarn ios
```

### Android

```sh
cd example
yarn android
```
